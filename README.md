# ML Engineering Lab

30 independent ML experiments spanning generative AI, NLP, computer vision, MLOps, and agentic systems. Built from scratch during BCA (2023–2026) to understand how these systems work under the hood before applying them at scale.

> **Next step:** Production systems in [ml-systems](https://github.com/neonite-rc/ml-systems).

## Experiments by Area

| Area | Count | Notable Projects |
|------|-------|-----------------|
| **LLMs & NLP** | 10 | RAG (6 variants), fine-tuning (LoRA/QLoRA), transformer from scratch, text summarization |
| **Generative AI** | 5 | Diffusion model from scratch, GANs, synthetic data generation, AI image generation |
| **MLOps & Deployment** | 4 | Airflow pipelines, Docker containerization, automated data cleaning (Rust), CI/CD |
| **Computer Vision** | 4 | CLIP alignment, drowsiness detection, image generation, geospatial clustering |
| **Agentic Systems** | 3 | Multi-agent orchestration, real-time AI assistant, agentic RAG router |
| **Data Engineering** | 4 | Feature engineering pipelines, ETL with Pandas, topic extraction, text classification |

## Projects Worth Reading
